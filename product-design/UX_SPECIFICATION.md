### Navigation Rules

**1. Dashboard is Home**
- Always one tap away from any screen
- Tapping ğŸ  icon always returns to Dashboard
- Shows fresh data on return

**2. Deep Linking**
- Each screen has a URL: `/dashboard`, `/journal`, `/fitness`, etc.
- Can bookmark specific screens
- Direct access via URL if logged in

**3. Floating Action Button (FAB)**
- Always visible: [+] button in bottom-right corner
- **iPhone:** Floats above content
- **iPad/Mac:** Fixed position bottom-right
- Single tap â†’ Quick Capture dialog
- Available from all screens except during active workout

**4. Back Navigation**
- **Mobile:** Back arrow top-left OR swipe from left edge
- **Desktop:** Back arrow top-left OR browser back button
- Always returns to previous screen (respects history stack)

**5. Modal Dialogs**
- Quick actions open as modals (Add Event, New Task, Quick Capture)
- Close via [X] button, tap outside, or ESC key (desktop)
- Preserve state if navigated away accidentally
